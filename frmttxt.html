Text formatting:

Text formatting in HTML refers to the process of applying various styles and attributes to text content to control its appearance on a web page. HTML provides several elements and tags that allow you to format text in different ways.

Example:

  • This text is bold


  • This text is italic


  • This is subscript

Heading text

You can use heading tags <h1> to <h6> for different levels of headings. These tags automatically format text to be larger and bold.

<h1>This is a Heading 1</h1> <h2>This is a Heading 2</h2>

Bold and Italic:

Use the <b> tag for bold and <i> tag for italic text.

<p><b>This is bold text</b></p> <p><i>This is italicized text</i></p>

Underline:

You can use the <u> tag to underline text.

<p><u>This is underlined text</u></p>

Font Size:

You can change the font size using the <font> tag and its size attribute.

<p><font size="5">This is larger text</font> </p>

Font Color:

You can change the text color using the <font> tag and its color attribute.

<p><font color="red">This is red text</font></p>

Alignment:

You can align text using the align attribute within various HTML elements.

<p align="center">This is centered text</p>

Keep in mind that while you can achieve basic text formatting using these HTML tags and attributes, using CSS is recommended for more advanced and consistent styling, as it separates the content from presentation and allows for greater control and flexibility in styling your web page. 

<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>
<p><strong>This is bold text</strong></p>
<p><em>This is italicized text</em></p>
<p><u>This is underlined text</u></p>
