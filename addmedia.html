Adding media(image,audio, video) to HTML:

HTML does provide various elements for embedding media content like images, audio, and video into web pages.

Adding Images in HTML:

Prepare Your Image: First, ensure you have the image you want to add to your web page. Make sure it's in a web-compatible format like JPEG, PNG, or GIF.

Create an HTML Document: Start by creating an HTML document or opening an existing one using a text editor or an integrated development environment (IDE).

Use the <img> Element: To add an image to your HTML document, use the <img> element.
Here's the basic structure of the <img> element:

<img src="image.jpg" alt="Description of the image">
src: This attribute specifies the source (file path or URL) of your image.

alt: This attribute provides a text description of the image.

It's essential for accessibility and SEO. Replace "Description of the image" with a brief, descriptive text.Insert Image into Your HTML: Place the <img> element within the HTML where you want the image to appear. For example: 
 <!DOCTYPE html>
<html>
<head>
    <title>My Web Page</title>
</head>
<body>
    <h1>Welcome to My Web Page</h1>
    <img src="image.jpg" alt="A beautiful landscape">
</body>
</html>
Save and View: Save your HTML file and open it in a web browser to see the image displayed on your web page.

Adding GIFs in HTML:

Adding a GIF is essentially the same as adding an image, as GIFs are also image files. Follow the same steps mentioned above for adding images, but ensure that you use a GIF file as the image source (specified in the src attribute).Here's an example of how to add a GIF to your HTML: 

<!DOCTYPE html>
<html>
<head>
    <title>My Animated GIF</title>
</head>
<body>
    <h1>Check out this cool animated GIF:</h1>
    <img src="animated.gif" alt="A funny animated GIF">
</body>
</html>
Replace "animated.gif" with the actual file path or URL of your GIF.Remember to provide meaningful alternative text (the alt attribute) for both images and GIFs to ensure accessibility and better SEO. This text is displayed if the image or GIF cannot be loaded or for users with disabilities who rely on screen readers.

Adding Audio in HTML:

Prepare Your Audio File: First, make sure you have the audio file you want to add to your web page. Common audio formats supported by web browsers include MP3, WAV, and OGG.

Create an HTML Document: Start by creating an HTML document or opening an existing one using a text editor or an integrated development environment (IDE).

Use the <audio> Element: To add audio to your HTML document, use the <audio> element. Here's the basic structure of the <audio> element: 

   <audio controls>
     <source src="audio.mp3" type="audio/mpeg">
     Your browser does not support the audio element.
   </audio>
<audio>: This is the main <audio> element that wraps the audio content.

- controls: The controls attribute adds audio playback controls (play, pause, volume, etc.) to the audio player.

- <source>: Inside the <audio> element, you can include one or more <source> elements. Each <source> element specifies a different audio file format and its source using the src attribute. The type attribute indicates the MIME type of the audio file.

- The text inside the <audio> element is a fallback message that will be displayed if the browser does not support the audio element.

Insert Audio into Your HTML:Place the <audio> element within the HTML where you want the audio player to appear. For example: 
   
   <!DOCTYPE html>
   <html>
   <head>
       <title>Audio Example</title>
   </head>
   <body>
       <h1>Listen to this audio:</h1>
       <audio controls>
           <source src="audio.mp3" type="audio/mpeg">
           Your browser does not support the audio element.
       </audio>
   </body>
   </html>
Save and View: Save your HTML file and open it in a web browser to see the audio player with playback controls. Users can click the play button to listen to the audio.

Remember to replace "audio.mp3" with the actual file path or URL of your audio file. The <audio> element provides a straightforward way to add audio content to your web pages, making it easy for visitors to listen to sound or music directly on your site.

Adding Video in HTML:

Prepare Your Video File: First, make sure you have the video file you want to add to your web page. Common video formats supported by web browsers include MP4, WebM, and OGG.

Create an HTML Document: Start by creating an HTML document or opening an existing one using a text editor or an integrated development environment (IDE).

Use the <video> Element: To add video to your HTML document, use the <video> element. Here's the basic structure of the <video> element:

<video controls width="640" height="360"> <source src="video.mp4" type="video/mp4"> Your browser does not support the video element.</video>
<video>: This is the main <video> element that wraps the video content.controls: The controls attribute adds video playback controls (play, pause, volume, etc.) to the video player.width and height: You can specify the dimensions (in pixels) of the video player using the width and height attributes.

<source>: Inside the <video> element, you can include one or more <source> elements. Each <source> element specifies a different video file format and its source using the src attribute. The type attribute indicates the MIME type of the video file.The text inside the <video> element is a fallback message that will be displayed if the browser does not support the video element.Insert Video into Your HTML: Place the <video> element within the HTML where you want the video player to appear. For example: 
 <!DOCTYPE html>
<html>
<head>
    <title>Video Example</title>
</head>
<body>
    <h1>Watch this video:</h1>
    <video controls width="640" height="360">
        <source src="video.mp4" type="video/mp4">
        Your browser does not support the video element.
    </video>
</body>
</html>
Save and View: Save your HTML file and open it in a web browser to see the video player with playback controls. Users can click the play button to watch the video.Remember to replace "video.mp4" with the actual file path or URL of your video file. The <video> element provides an easy way to add video content to your web pages, allowing visitors to watch videos directly on your site with built-in playback controls.
