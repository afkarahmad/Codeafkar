Some properties of CSS:

Here are some useful CSS property

1.color:

The CSS color property lets you define the text color of HTML elements. For instance: 
 
 h1 {
    color: blue; /* Sets <h1> text color to blue. */
}

a {
    color: #FF0000; /* Makes links red using hexadecimal notation. */
}

p {
    color: rgb(0, 128, 0);
    /* Changes <p> text to green with RGB notation. */
}
In these examples, the color property is applied to headers, links, and paragraphs, allowing you to control the text color for each element individually.

2.font-size:

The font-size property sets the size of text within HTML elements. It allows you to specify the desired font size using units like pixels (px), em, rem, or percentages (%). For example: 
h1 {
    font-size: 24px; /* Sets the font size of <h1> elements to 24 pixels. */
}

p {
    font-size: 1.2em; /* Makes text in <p> elements 20% larger than their parent's font size. */
}

a {
    font-size: 16px; /* Specifies a font size of 16 pixels for links. */
}
This property is essential for adjusting the appearance and readability of text on a web page to match your design requirements.
